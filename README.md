# Windows 注册表

记录一些 Windows 注册表的信息，比如[右键菜单](context-menu)。

## 注意事项

仓库所有文件使用 UTF-8 编码，部分出现中文的注册表在中文 Windows 系统中可能出现乱码，请使用任意文本编辑工具将文件编码转换为 GBK。
